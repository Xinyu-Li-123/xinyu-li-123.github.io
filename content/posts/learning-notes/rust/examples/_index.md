One good way to learn something is learn by example. This is what the famous [Rust by Example](https://doc.rust-lang.org/rust-by-example/) is about, which explains individual Rust concepts by examples. In this series I also collects some of the Rust code snippets I find that touch multiple Rust concepts at once, showing how they work together.
